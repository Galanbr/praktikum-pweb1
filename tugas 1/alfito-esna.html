Praktikum Pemrograman Web 1

Praktikum Pemrograman Web memberikan Keterampilan kepada mahasiswa dalam pembuatas situs web yang dinamis dan dapat berinteraksi dengan user Materi meliputi Teknologi Web,Pemrograman situs HTML, CSS, PHP, MySQL dan Visual Studio Code.

Keterangan:

  1. Buatlah class bernama .dashed untuk heading utama
  2. Gunakan selector div untk membuat Box Model
  3. Buatlah class bernama underline-red untuk kalimat dengan garis bawah merah
  4. Buatlah class bernama shadow untuk memberikan efek bayangan pada teks
